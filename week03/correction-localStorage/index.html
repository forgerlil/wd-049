Local Storage crash course

Check out your local storage in the Application tab of the browser's console (Or the Storage tab if you're on Firefox)!

The Web Storage API allows you to save {key: value} pairs in your browser's cache. Local Storage data persists after the leaving the website (it is bound to the url, so whenever you return to the same url your localStorage will load up all data again). It is a property of your Window global object, like the document.

For this exercise you'll have to:
  1. Add items to localStorage with the "Store me" button
  2. Remove one specific item from localStorage with the "Clear One" button
  3. Show all items in localStorage with the "Show Storage" button
  4. Clear localStorage with the "Clear All" button
