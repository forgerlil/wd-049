ES6 Modules: Basics

In its early days, Javascript programs were pretty small. Over time, splitting code into separate files became a necessity. Since 2015, ES6 has introduced modules natively to JavaScript.

Your goal: Split code into different files with modules.
  • You will need 3 JS files for this exercise:
    1. index.js is your main module. It should only log or call structures imported from other files. Create it, and add it to your HTML file as a module.
    2. myChoices.js is where you'll store values about yourself. Create the file. It can stay blank for now.
    3. computerChoices.js contains a helper function. It has already been provided to you.
  • In myChoices.js:
    1. Create a variable containing your favorite quote as a string. Export it by default. In index.js, import it and log it.
    2. Create an object to represent your favorite movie or book. Export it as a named export. In index.js, import it and log it.
    3. Create a function to play Odds and Evens. It should accept three arguments: two numbers and a string with your choice for "odd" or "even", and return a message declaring the result. Export it as a named export.
  • In computerChoices.js:
    1. You were given a function that generates a random number. As a named export, export it in a different way then what you used so far.
  • In index.js:
    1. Import the function from computerChoices.js. Try to use a different syntax for importing!
    2. Import your function from myChoices.js. Call it with a number generated from the function from computerChoices.js for the first parameter, your choice between 0 and 5 as the second parameter, and your choice of "odd" or "even" as the third parameter.
